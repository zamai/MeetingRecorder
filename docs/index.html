MeetingRecorder

Capture system audio and microphone input in one click

A lightweight macOS menu bar app that records both system audio and your microphone simultaneously, merging them into a single M4A file.

Download Source Code

Features

Dual Recording

Capture system audio and microphone input at the same time with a single click.

Menu Bar App

Lives in your menu bar for quick access. Start and stop recording instantly.

M4A Output

Records to high-quality M4A (AAC) format for excellent compression and compatibility.

Auto Merge

Automatically merges system audio and microphone tracks into a single file.

Requirements

  • macOS 14.0 (Sonoma) or later
  • Microphone access permission
  • Screen recording permission (for system audio capture)
